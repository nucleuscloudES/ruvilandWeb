# RUVILAND Web

Esta es la web oficial del servidor de Minecraft **RUVILAND**.

## Propósito
La web está diseñada para servir como punto de acceso rápido a los enlaces de interés de la comunidad, como Twitch, Discord y descargas de modpacks. Además, hospeda imágenes que pueden ser utilizadas para otras funciones relacionadas con el servidor y la comunidad.

## Estructura
- `index.html`: Página principal con enlaces y presentación.
- `css/stylesheet.css`: Estilos personalizados para la web.
- `img/`: Carpeta de imágenes usadas en la web y para otros propósitos.
- `js/`: Scripts para efectos visuales y funcionalidad adicional.

## Enlaces principales
- Twitch de los streamers.
- Discord de La Olla.
- Descarga de modpacks recomendados.
